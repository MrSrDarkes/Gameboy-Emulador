# Carpeta ROMs

Coloca aquí tus ROMs para jugar en el emulador.

**Formatos soportados:**
- **.gba** — Game Boy Advance (ej: Pokémon Esmeralda, Rojo Fuego)
- **.nds** — Nintendo DS (ej: Pokémon Negro, Platino, Heart Gold)

**Ejemplos de nombres** (puedes usar los que quieras):
- `Pokemon - Edicion Esmeralda (S).gba`
- `Pokemon - Edicion Rojo Fuego (S).gba`
- `Pokemon Negro (S).nds`
- `Pokemon Edicion Platino ESP.nds`
- `Pokemon Heart gold_esp.nds`

Luego usa **"Cargar ROM"** en el emulador y selecciona el archivo desde esta carpeta.
